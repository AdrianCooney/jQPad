Usage

Usage of this framework is just editing the index.html and other aspects of the already created template and sort-of demo. In more releases to come, it will be widely customizable!

FAQ

Basic

These a jQPad's optional parameters.

$.jQPad({
siteName: "jQPad",
defaultHomeTitle: "Home",
defaultPage: "content/default/",
ajax: {
cache: true,
defaultNavMethod: true
},
splash: "misc/jqpad-splash.png",
icon: "misc/jqpad-icon.png",
statusbarColor: "white"
});

Site Name

Site name is, obviously, the site's name.

Default Home Title

This is the title of the default page when the site loads.

Default Page

This is the url location of the default page to be loaded on site load. Usually in default/.

Bookmark icon

The URL for the homepage icon.

Splash

The splashscreen is the 'loading' screen when the user adds the site to their homescreen.

StatusBar color

The color of the status bar, either black or white.

The AJAX.

The jQPad works by taking data from each link and options specified and displaying it accordingly. Its several types of attributes are the placeholders for its 'link data'.

Example:

<a href="#general/" title="General Settings" data-call="type=4&awesome=true">General</a>

href attribute

<a href="#general/" title="General Settings" data-call="type=4&awesome=true">General</a>

Every link within the framework has to start with a hash tag(#) unless the link is to make the page navigate away from the site. The data after the hash tag(#) is the location for the AJAX to load the content. You need to store your content in the content/ directory. You can also add data to the end for GET'ting by PHP e.g '#general/index.php?type=4&awesome=true'

NOTE:
you do not to include content/ when specifying the location to load the content. it automatically adds that for you.

title attribute

<a href="#general/" title="General Settings" data-call="type=4&awesome=true" >General</a>

The title attribute is the data that the AJAX takes and uses to display the title above each page in the toolbar. There is no specifications for this attribute, only that you include it in each link.

data-call attribute

<a href="#general/" title="General Settings" data-call="type=4&awesome=true" >General</a>

The data-call is for POST'ing data via an AJAX call. jQuery manipulates it for parsing later on by PHP or Javascript etc.


Features

Share content

In today's world, sharing is imperative for success, with jQPad, it becomes a breeze to add your page to Digg or Twitter by just adding #share in the href attribute of a link. When clicked, a sleek share bubble pops under your "Share Me!" link.

<a href="#share">Click me to share!</a>

Make lists deletable

The iPad's swipe to delete is now very easily do-able with a simple jQPad function! To make lists 'deletable', you add .deletable(); to your list.

Example:

$("li").deletable(function() { /* Delete from database */});

Panels

It opens a panel anywhere on the site, all you have to do is direct it to it! It will show it, expand it and automatically add a close button, handy eh? To use, add the panel id after #panel-.

Example:

<a href="#Foobar" data-target="#panel-PANELID">Open My panel!</a>

To number lists.

To number lists with a customizable bubble beside each list item is also possible via a simple jQPad function, .numbered()

Example:

$("li").numbered();

NOTE:
you cannot use both on the same list. its is a known bug and would only make things slow.
